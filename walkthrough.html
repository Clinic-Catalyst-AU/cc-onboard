The one spot you update each month - your booking link

Whenever your offer changes, you update your booking link in one place in GHL. Everything that hands out a link reads from this spot, so you never edit the AI agent or the emails.

In GHL: Settings  →  Custom Values  →  booking_link  →  paste this month's booking link  →  Save

That single change updates both:

  • Your AI agent - the link it gives anyone who wants to book
  • Every nurture + newsletter email - they all pull the same link

Tip: make sure the booking_link value is filled for your contacts too, so every message points to the right place.

Just landed - your end-of-day shutdown skill

/cc-shutdown - say "shutdown" (or "wrap up", "done for the day") at the end of any Claude session. It reports everything touched as DONE / NEEDS YOU / IN PROGRESS, catches anything half-finished (unapproved drafts, imports mid-run), and saves what matters - so tomorrow starts clean instead of confused.
