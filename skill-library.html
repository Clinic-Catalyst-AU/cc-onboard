Clinic Catalyst - Skill Library

Every tool in your system, what it does, and how to run it

How to run any skill: open Claude Code, type the command (e.g. /cc-landing-page) and press enter. Claude asks for anything it needs. That's it - you talk, it builds. Everything reads your Business Brain so it already sounds like your clinic.
Prefer to watch? Every skill below is shown on video - what it does and exactly what to type - in the Video Library.
Set up & keep current
/cc-resonance

Interviews you and writes your messaging foundation - the emotional source of truth every other skill reads from.

Run when: first, before anything else.

/cc-brand-guide

Turns your resonance doc into a clean brand guide so every output sounds like you.

Run when: straight after /cc-resonance.

Content & reels
/cc-content-engine

Builds a full month of on-brand, AHPRA-safe social posts from your offer strategy.

Run when: planning the month's content.

/cc-find-clip

Finds the single best ~2-minute clip from a longer video, ready to post.

Run when: you filmed something long.

/cc-reeltalkinghead

Raw talking-to-camera video in, captioned approval-ready reel out (cuts the dead air too).

Run when: you filmed yourself talking.

/cc-reelassembly

Edits clips in a told order (walk in, meet you, explain, call to action) with captions.

Run when: you have a story to tell in clips.

/cc-reelclips

Quick montage for a trending-audio reel - you add the sound in-app.

Run when: jumping on a trend.

/cc-carousel

Branded multi-slide carousel posts.

Run when: teaching or listing in slides.

/cc-reelvoiceover

Writes and records a narration over your b-roll, cut to the footage.

Run when: you have clips but no one talking.

/cc-assemble

Stitches clips together into one video, with music that ducks under any talking.

Run when: joining clips into a montage or reel.

/cc-cover

A branded reel/story cover in your accent colour and font - photo, headline, one accent word.

Run when: a reel needs a cover.

/cc-hook

Sharpens the opening line of any post or reel so people stop scrolling.

Run when: the opener isn't landing.

Run when: optimising pages to get found.
