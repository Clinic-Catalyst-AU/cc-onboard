The one shift everything hangs off

The agents are already built.

Up to now we've built each AI agent from scratch in the room - the marketing agent, the ad agent, the offer agent, the video agent, the compliance agent. That's the slow way, and it's the way you know because you haven't had the system yet.

The whole point of the AIOS is that those agents ship ready-made. Every clinic installs a pack of skills on Day 1. So in the room we do not build them - we run them, by typing one command. Hours of building collapse into minutes of running.

So three things reorder
1

The install runs in the background from 9:00

The full install (Claude Code + the skill pack + the tools) takes 30-60 minutes. So we kick it off the moment everyone sits down, and it churns quietly while you run Welcome + CRM.

Was: "Claude account setup" as a quick 11am task. Now: started at 9:00, done by morning tea - or we lose the afternoon.

2

We build the Business Brain first

Every skill reads one foundation - the clinic's Business Brain (their voice, their offer, their clients' real emotional truth). Build that first and every skill produces on-brand, compliant copy automatically.

Was: "Marketing Agent Build", "Offer Creation", "Neuro-Emotive Framework" as separate blocks. Now: those are all just - build the Business Brain, then run the skills.

3

The framework is branded Clinic Catalyst

Your neuro-emotive method is brilliant and it stays - it runs inside the resonance skill. It's just branded Clinic Catalyst in this workshop, because AI With Soul is a separate brand and can't appear in a Clinic Catalyst room.

Was: "using Alicia's AI With Soul Framework". Now: the Clinic Catalyst resonance method (same framework, our brand).

What I'm doing on my end

I'm installing the complete system on my machine first

Some of this is quicker on my end than it'll be in the room. The full install takes me about an hour, and then we'll trial it working together - so by the time we walk into the workshop, we've both seen the whole flow run end to end. No surprises, no figuring it out live.
