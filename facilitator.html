Your step-by-step for onboarding Alicia + the crew, then teaching the resonance to brand-guide flow.

They open the download page; you drive from here. Their page: cc-onboard (install) try.html (practice)
Phase 0 · Before the day (you, ~2 min)

One thing from your machine.

1 Invite them to the GitHub org

Gives them the private cc-aios repo + all CC client repos. Swap in their email. That's the only prep.

gh api -X POST orgs/Clinic-Catalyst-AU/invitations -f email=THEIR_EMAIL -f role=direct_member
Phase 1 · Get their machine set up (with them, ~20 min)

Send them to the install page and walk alongside. It is mostly hands-off (one bootstrap command).

2 They run the download page

They follow cc-onboard top to bottom: install the 4 apps → gh auth login → the one-line bootstrap (installs Node/Python/ffmpeg/Claude Code/Playwright + all CC skills). ~10-15 min, grab a tea.

3 The 2 things the script can't do (do these live)

Keys: open their ~/Systems/BusinessOps/.env and type in the SHARED keys (GHL, CC Supabase, Telegram) - in person, never over chat. Gemini/OpenAI/Fireflies they use their own.

Connectors: in their Claude Code, connect Fireflies, Supabase, GitHub (OAuth, their own logins). Playwright MCP is already wired by the bootstrap.

Phase 2 · Teach the flow - the value moment (with them)

This is what they are really here for: how to take a clinic from nothing to an on-brand Claude.

The chain: /cc-resonance → resonance-messaging.md → /cc-brand-guide → brand-guide.md → paste into the clinic's Claude. Run it live on a real or sample clinic so they see it work.

4 Run the resonance intake

In Claude Code, with a clinic (or a sample), run it and walk through the 13 questions. The being-deeply-understood feeling IS the magic - let them feel it.

/cc-resonance [Clinic Name]

Produces their resonance-messaging.md in the shared CC folder. (Don't share the framework - only the output.)

5 Generate the brand guide

It reads the resonance doc, asks only the few identity gaps (name, tagline, colours, vibe), and writes the paste-into-Claude file.

/cc-brand-guide [Clinic Name]

Produces brand-guide.md. They paste its Claude Context Block into the clinic's Claude Project custom instructions - now every output sounds on-brand + AHPRA-safe.

6 Let them play

Point them at the trial page and let them fire prompts - /cc-prospect, /cc-ad-builder, /cc-pitch.

Open try.html
