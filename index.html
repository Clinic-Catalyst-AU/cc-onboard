1 Get the basics

Install Homebrew + the GitHub CLI in one line, and grab the two apps.
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" && brew install gh
Download Dropbox (sign in + accept Kelly's shared folder)

2 Sign in to GitHub

Pick: GitHub.com, then HTTPS, then login with a browser.
gh auth login && gh auth setup-git

3 Install the whole system (one command)

This sets up everything the skills need - Node, Python, ffmpeg, Claude Code, Playwright, the video/reel engine, all the CC skills, and your config.
~25-45 min on a fresh Mac - start it FIRST and let it run in the background
curl -fsSL https://clinic-catalyst-au.github.io/cc-onboard/bootstrap.sh | bash

Kick this off first thing - it downloads a lot. Leave it running while we do the welcome + strategy session, and check back when it prints DONE. If it ever says DONE WITH PROBLEMS, screenshot it and text Kelly.

4 Add the keys

Kelly sends you the shared keys (GHL, Supabase, Telegram) securely - paste them in. Open the file:
open -e ~/Systems/BusinessOps/.env

Paste each value after its =, save, close. (The "OWN" keys you can add anytime - they're only needed for image/transcription skills.)

5 Start Claude Code (your account)

Close + reopen Terminal first, then:
claude

Log in with your own Anthropic account when prompted.

You're in. Now watch this.

In Claude Code, type:

/cc-prospect

Paste a clinic's website and watch the whole prospect → demo → deck → deploy pipeline run - AHPRA-compliant, on-brand, on your machine. You now have everything Kelly has for Clinic Catalyst.
