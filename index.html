Clinic Catalyst

Welcome aboard, Alicia

Five steps and you'll have the entire Clinic Catalyst system on your Mac.

Before you start: Kelly will have (1) invited you to the Clinic-Catalyst-AU GitHub org - accept the email, and (2) shared the Clinic_Catalyst_Shared Dropbox folder - accept that too.

1 Install the 4 apps

Skip any you already have. Click to download.
Claude Code Dropbox Homebrew GitHub CLI

Prefer the terminal for the last two? Paste this (installs Homebrew, then the GitHub CLI):

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" && brew install gh

Then sign in to Dropbox and accept Kelly's shared folder, and open Claude Code once so it finishes setup.

2 Sign in to GitHub

Connects your machine to the Clinic Catalyst code. Choose: GitHub.com, then HTTPS, then login with a browser.
gh auth login && gh auth setup-git

3 Install Clinic Catalyst

The magic line - pulls the system and installs every CC skill into Claude Code.
gh repo clone Clinic-Catalyst-AU/cc-aios ~/Systems/cc-aios && ~/Systems/cc-aios/bin/apply-to-machine.sh

4 Check it worked

You should see the CC skills and the shared client folder.
ls ~/.claude/skills | grep -E 'cc-|aria|timely' ls ~/Dropbox/Clinic_Catalyst_Shared/Clients

5 Accept the shared workspace

In Dropbox, accept Kelly's Clinic_Catalyst_Shared invite. That folder is all our CC client work, synced between us automatically. Just save into it and it shows up for both of us.

You're in. Now watch this.

Open Claude Code and type:

/cc-prospect

Paste a clinic's website and watch the whole prospect to demo to deck to deploy pipeline run - AHPRA-compliant, on-brand, on your machine. You now have everything Kelly has for Clinic Catalyst.
