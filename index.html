1 Get the basics

Install Homebrew + the GitHub CLI in one line, and grab the two apps.
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)" && brew install gh
Download Dropbox (sign in + accept Kelly's shared folder)

2 Sign in to GitHub

Pick: GitHub.com, then HTTPS, then login with a browser.
gh auth login && gh auth setup-git

3 Install the whole system (one command)

This sets up everything the skills need - Node, Python, ffmpeg, Claude Code, Playwright, the video/reel engine, all the CC skills, and your config.
~25-45 min on a fresh Mac - start it FIRST and let it run in the background
curl -fsSL https://clinic-catalyst-au.github.io/cc-onboard/bootstrap.sh | bash

Kick this off first thing - it downloads a lot. Leave it running while we do the welcome + strategy session, and check back when it prints DONE. If it ever says DONE WITH PROBLEMS, screenshot it and text Kelly.

One catch: if Dropbox was still syncing when this ran, it prints a note about the shared folder. No drama - once Dropbox finishes, just paste this same command again. Second pass links your ~/Clients to the shared client folders (the pitch skill needs that).

4 Add the keys (with Kelly, in person)

Two files get the shared values typed in - Kelly has them. This happens on BOTH machines (these files never sync, on purpose).

4a. The .env file - GHL, Supabase, Telegram + the shared Gemini key:

open -e ~/Systems/BusinessOps/.env

Paste each SHARED value after its =, save, close. The Gemini key is Kelly's (shared for now) - check with her before any big image-generation batches. The "OWN" keys you can add anytime.

4b. The GHL connector token - same GHL key (CC_GHL_PIT), one more home:

open -e ~/.claude/settings.json

Find "gohighlevel" and paste the GHL key into BOTH empty values - BEARER_TOKEN_BEARERAUTH and BEARER_TOKEN_BEARER. Keep the quotes, save, close.

5 Start Claude Code (your account)

Log in with your own Anthropic account when prompted. Then connect your own Fireflies, Supabase and GitHub connectors when Claude offers them (your logins, per machine).
